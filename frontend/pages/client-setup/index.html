2. Customize Client Branding

Each client dashboard can be customized with:

  • Custom company name and logo
  • Brand colors and themes
  • Business hours and timezone
  • Industry-specific messaging
  • Custom phone numbers

3. Client Dashboard URL

Each client gets their own unique dashboard URL:

https://ai-booking-mvp.onrender.com/client-dashboard?client=CLIENT_KEY

Replace CLIENT_KEY with your client's unique identifier.

4. Add New Clients

To add a new client, update the CLIENT_CONFIG object in the client dashboard file with:

'new_client': {\n name: 'New Client Name',\n description: 'Client description',\n logo: '🏢',\n color: '#2196F3',\n bg: 'linear-gradient(135deg, #2196F3 0%, #1976D2 100%)',\n phone: '+447000000000',\n timezone: 'Europe/London',\n businessHours: '9AM - 5PM, Mon-Fri',\n industry: 'Professional Services',\n website: 'newclient.com'\n}

💡 Pro Tips:

  • Use industry-specific logos and colors for better branding
  • Set up proper business hours for each client's timezone
  • Customize the description to match the client's business
  • Test the SMS functionality with each client's phone number
