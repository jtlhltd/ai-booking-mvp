## üìã TRACKING SHEET STRUCTURE

### Column Definitions:

**IDENTIFICATION:**
- **Date** - When you contacted them
- **Name** - Decision maker's name
- **Practice** - Dental practice name
- **Email** - Contact email
- **Phone** - Contact phone (if available)

**OUTREACH DETAILS:**
- **Channel** - Email, LinkedIn, LinkedIn InMail, Call, or Combination
- **Template** - Which template/script you used (e.g., "ROI Hammer", "Time Saver", "InMail Direct")
- **Contact_Number** - 1st contact, 2nd (follow-up), 3rd (final), etc.

**RESPONSE TRACKING:**
- **Response** - Yes/No/Partial
- **Response_Date** - When they responded
- **Outcome** - NEW, CONTACTED, RESPONDED, DEMO_BOOKED, DEMO_COMPLETED, TRIAL_STARTED, CLIENT, LOST

**METADATA:**
- **Lead_Source** - Where you found them (Google Maps, LinkedIn, Referral, etc.)
- **Notes** - Any learnings, objections, or details

## üîç HOW TO USE TRACKING FOR IMPROVEMENT

### Week 1: Establish Baseline

**Goal:** Send consistent volume across all channels

**Actions:**
- Send 50 emails/day (using Template 1)
- Send 30 LinkedIn/day (using Template 1)
- Make 20 calls/day (using Call Script 1)
- Track EVERYTHING in the sheet

**End of Week 1:** Calculate baseline metrics
- Email response rate: X%
- LinkedIn response rate: Y%
- Call response rate: Z%

---

### Week 2-3: A/B Test ONE Variable

**Goal:** Test different templates on the SAME channel

**Experiment Setup:**
```
Group A (100 emails): Template "ROI Hammer"
Group B (100 emails): Template "Time Saver"

Keep everything else the same:
- Same time of day
- Same day of week
- Same prospect type
- Same personalization level
```

**Track Results:**
- Group A response rate: _____
- Group B response rate: _____
- Winner: _____

**Action:** Use the winning template going forward

---

### Week 4: Test Channels

**Goal:** Determine which channel to prioritize

**Compare:**
- LinkedIn vs Email response rates
- Which takes more time per touch?
- Which converts to demos better?

**Decision:**
```
If LinkedIn 2x better response BUT takes 3x longer ‚Üí Focus on Email
If Calling highest conversion ‚Üí Do calling on warm leads only
```

---

### Week 5+: Refine & Scale

**Goal:** Optimize what works

**Actions:**
- Double down on highest-converting template
- Do more of your best channel
- Eliminate what doesn't work
- Test small tweaks (subject lines, CTAs)

**1. MEASURE** (Friday)

- Run the experiment

- Collect data

**5. DECIDE** (Friday)

- Did it fail? Kill it
- Inconclusive? Test more
