# Plans (persisted)

Plans live in this folder so they survive app reloads and can be referenced from commits/PRs.

## Naming

- `YYYY-MM-DD_<short-kebab-slug>.md`

## Template (minimum)

- Context
- Definition of done
- Non-goals
- Work breakdown (checklist)
- Risk & rollback
- Amendments (append-only, if the plan changes mid-flight)
